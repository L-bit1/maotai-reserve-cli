## 一、说明

### 1.1 计价原则

| 项目 | 说明 |
|------|------|
| 计价单位 | **人天**（1 人天 = 8 小时有效开发） |
| 人天单价 | **1,000～1,500 元/人天**（中级工程师，含设计、开发、自测、文档） |
| 费用区间 | 人天 × 单价，取整为 **万元级区间** 便于合同填写 |
| 不含项 | 云服务器、代理 IP、短信、PushPlus、高德 API、税费、差旅 |

### 1.2 评估依据

- 当前仓库：`src/`（一期 CLI）、`backend/`（二期 API 约 50%～60%）、`web/`（管理端约 70%）
- 需求范围：`docs/前后端接口方案.md` 分期（M1～M4）
- 进度基线：`docs/进度报告-五周计划.md`（2.5 周开发 + 2.5 周调试）

### 1.3 重要声明

- 本工具为 **i茅台预约 + 抽签** 辅助，**不保证中签率**。
- 合同 KPI 建议写 **预约提交成功率**、**账号在线率**，勿写「中签率 67%/20%」等。
- 1000 账号同网段需 **IP 隔离**（代理/热点），运维成本另计。

## 十一、压缩工期加价（甲方赶工参考）

**原则**：压缩的不是「写字速度」，而是 **并行人力 + 加班 + 测试被挤压的风险**；必须 **书面约定范围不减、验收标准不降**。

### 11.1 基准工期（本项目的合理节奏）

| 范围 | 正常工期 | 说明 |
|------|----------|------|
| 二期 Web + API 补全 | **3～4 周** | 联调、部署、文档 |
| 全量 + 验收（套餐 C） | **5～6 周** | 含 2.5 周调试压测 |
| 1000 号 IP 方案落地 | **+1～2 周** | 依赖甲方提供代理资源 |

### 11.2 加价系数（在「原合同开发费」上加，不含服务器/代理）

| 压缩幅度 | 示例 | 建议加价比例 | 适用 |
|----------|------|--------------|------|
| 轻度 | 6 周→5 周（约 -17%） | **+10%～15%** | 仅减少缓冲，加班少 |
| 中度 | 6 周→4 周（约 -33%） | **+25%～35%** | 需周末/并行，测试减样 |
| 重度 | 6 周→3 周（约 -50%） | **+40%～60%** | 必须加人或砍范围 |
| 极限 | 6 周→2 周 | **+60%～100%** 或 **拒接全量** | 仅交付「可演示版」 |

> 行业常见：**每压缩约 20% 日历时间，开发费加价约 10%～20%**（非线性，越急越贵）。

### 11.3 按套餐 C 举例（原开发费约 8.5～12.6 万）

| 原工期 | 压缩后 | 建议加价 | 压缩后总价（约） |
|--------|--------|----------|------------------|
| 6 周 | 5 周 | +0.8～1.2 万 | **9.5～13.5 万** |
| 6 周 | 4 周 | +2.0～3.5 万 | **10.5～15 万** |
| 6 周 | 3 周 | +3.5～6 万 | **12～18 万** |

**仅二期套餐 B（原 4～5.3 万）**：

| 原工期 | 压缩后 | 建议加价 | 压缩后总价 |
|--------|--------|----------|------------|
| 4 周 | 3 周 | +0.5～0.8 万 | **4.5～6 万** |
| 4 周 | 2 周 | +1.0～1.8 万 | **5～7 万** |

### 11.4 必须写进合同的「交换条件」（否则不加价也吃亏）

1. **范围冻结**：压缩工期 = 不加新功能；变更走变更单另计费。  
2. **验收降级需明示**：如「压缩 3 周交付」则 **不含** 1000 号压测、不含 MySQL 生产迁移。  
3. **甲方配合时限**：代理 IP 列表、测试账号、域名备案 **逾期则工期顺延、不加价作废**。  
4. **付款节奏前置**：赶工建议 **50% 签约 / 30% 联调 / 20% 验收**，或加 **10% 赶工预付款**。  
5. **维保缩短**：压缩交付的质保建议 **7 天** 而非 15 天（P0 修 bug）。

### 11.5 谈价话术（给乙方）

> 「可以加快，但需要 **赶工费 XX 万**：因为要并行联调、加班排期，且调试时间被压缩，后期 429/多账号问题概率会上升。我们建议在 **4 周内交付套餐 B**，加价 **25%**；若必须 **3 周全量验收**，加价 **40%～50%**，且 1000 号压测改为二期或甲方自备环境后自行压测。」

### 11.6 不建议免费压缩的情况

- 要求 **中签率 / 成功率** 承诺不变  
- 要求 **1000 号 + 全功能 Web + 3 周** 同时满足  
- 不增预算但 **增加合同 (1)～(6) 新条目**
