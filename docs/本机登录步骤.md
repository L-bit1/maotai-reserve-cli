# 本机登录步骤（解决服务器 Web 无法发码）

服务器 IP 被 i茅台 429，**Web「发码/登录」会失败**。请在本机（Mac / 办公室电脑）运行脚本，Token 自动同步到服务器。

## 一、最快方式（Mac）

1. 打开文件夹：`茅台抢单软件`
2. **双击** `启动本机登录.command`
3. 用户名：`owner`
4. 密码：你的 **owner 管理密码**（勿写入公开文档或 GitHub）
5. 对每个号：等短信 → 输入 6 位验证码 → 回车
6. 输入 `s` 跳过当前号；每个号之间自动等 90 秒

## 二、终端方式

```bash
cd "/Users/mac/Desktop/工作 /软件/茅台抢单软件"

# 自检本机能否发码
.venv/bin/python scripts/test_local_vcode.py --mobile 13800000001

# 批量登录全部未登账号
.venv/bin/python scripts/batch_local_login.py --password '你的owner密码'

# 只登 id 1～3
.venv/bin/python scripts/batch_local_login.py --password 'xxx' --ids 1,2,3

# 单号
.venv/bin/python scripts/local_login_to_server.py --mobile 18785006433 --password 'xxx'
```

## 三、登录完成后

1. 打开 http://你的服务器/ → **账号管理**
2. 确认「登录」列为 **已登**
3. 点击 **同步凭证文件**
4. 配置预约任务

## 四、发码失败时

| 现象 | 处理 |
|------|------|
| 429 限流 | 换 **4G 热点**；或等 2 小时；或 `--proxy 'http://user:pass@host:port'` |
| 天启代理 645 | **不要用天启**，本机直连或换其他代理 |
| 已收到短信但发码报错 | 脚本会提示直接输入验证码 |

## 五、办公室电脑 / 轻量云

拷贝整个项目目录，安装依赖后运行 **同一命令** 即可：

```bash
python3 -m venv .venv
.venv/bin/pip install -r backend/requirements.txt
.venv/bin/python scripts/batch_local_login.py --password 'owner密码'
```

不需要高带宽；关键是 **出口 IP 能发码**（先用 `test_local_vcode.py` 测）。
