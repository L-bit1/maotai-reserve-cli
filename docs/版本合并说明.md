# 版本合并说明（龙蒙超版 + 本版）

合并前完整快照（不含 `.venv`、`reverse/jadx_out`）：

```

```

龙蒙超原始目录仍保留：

```
龙蒙超版本/maotai-reserve-cli-main/
```

## 合并策略

| 来源 | 保留内容 |
|------|----------|
| **本版** | 捡漏波次、低竞争选店、IP 分组、`proxy_pools`、登录 429 不重试、文档与脚本 |
| **龙蒙超版** | 随机 UA/网络类型、行为 `warmup`、预约错峰 `jitter`、概率领小茅运、随机重试间隔 |

## 新增配置 `antidetect`

见 `config.example.yaml`。若现有 `config.yaml` 无此段，程序使用默认值（反检测**开启**）。

关闭反检测（调试 / 对比旧行为）：

```yaml
antidetect:
  enabled: false
```

## 凭证字段兼容

- 本版：`proxy_url`、`egress_group`
- 龙蒙超版：`proxy` → 读取时自动映射到 `proxy_url`
