# 茅台抢单软件 — 前后端接口方案（二期）

**版本**：v0.1  
**日期**：2026-05-20  
**说明**：一期 CLI 逻辑保留在 `src/`；二期新增 **管理后端 API** + **Web 管理前端**，通过 HTTP 调用复用现有 Python 能力。

┌─────────────────────────────────────────────────────────────┐

│  - 账号 / 商品 / 门店 / 任务 / 记录 / 统计                    │

### 1.1 端口规划

| 服务 | 默认端口 | 协议 | 说明 |
|------|----------|------|------|
| **管理后端 API** | `8000` | HTTP | FastAPI，`/api/v1/*` |
| **API 文档** | `8000` | HTTP | Swagger `/docs`，ReDoc `/redoc` |
| **Web 管理前端** | `5173` | HTTP | Vite 开发服务器 |
| **前端生产** | `80` / `443` | HTTP(S) | Nginx 反代到 5173 静态 + `/api` → 8000 |
| **MySQL** | `3306` | TCP | 生产推荐 |
| **Redis**（可选） | `6379` | TCP | 任务队列 / 限流计数 |
| **PushPlus** | — | 外网 | 微信推送，出站 HTTPS |

### 1.2 技术选型建议

| 层级 | 选型 |
|------|------|
| 后端 | Python 3.10+、**FastAPI**、SQLAlchemy 2、Alembic |
| 数据库 | SQLite（开发）/ **MySQL 8**（生产） |
| 任务 | APScheduler（轻量）或 Celery + Redis（账号量大时） |
| 前端 | Vue3 + Element Plus 或 React + Ant Design |
| 鉴权 | JWT（管理端单用户或 RBAC） |
| 部署 | Docker Compose（api + web + mysql + nginx） |

## 二、统一约定

### 2.1 Base URL

```text
开发: http://127.0.0.1:8000/api/v1
生产: https://your-domain.com/api/v1
```

### 2.2 通用响应格式

**成功：**

```json
{
  "code": 0,
  "message": "ok",
  "data": { }
}
```

**失败：**

```json
{
  "code": 40001,
  "message": "验证码不能为空",
  "data": null
}
```

### 2.3 通用错误码

| code | 含义 |
|------|------|
| 0 | 成功 |
| 40100 | 未登录 / Token 失效 |
| 40300 | 无权限 |
| 40001 | 参数错误 |
| 40029 | 操作过频（对应 i茅台 429 映射） |
| 50000 | 服务器内部错误 |
| 50001 | i茅台 上游接口失败 |
| 50002 | 非申购时段 sessionId=0 |

### 2.4 鉴权

| 接口类型 | Header |
|----------|--------|
| 管理端登录后 | `Authorization: Bearer <jwt>` |
| 内部服务 | 可选 `X-Admin-Key`（单机部署） |

**不涉及** 把 i茅台 `MT-Token` 暴露给浏览器；Token 仅存服务端数据库加密字段。

## 三、管理后端 API 清单

### 3.1 认证 ` /auth `

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/auth/login` | 管理员账号密码登录，返回 JWT |
| POST | `/auth/refresh` | 刷新 Token |
| GET | `/auth/me` | 当前管理员信息 |

**POST `/auth/login` 请求：**

```json
{
  "username": "admin",
  "password": "******"
}
```

**响应 data：**

```json
{
  "access_token": "eyJ...",
  "expires_in": 86400
}
```

### 3.3 i茅台账号 ` /accounts `（合同 (1)）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/accounts` | 分页列表（脱敏手机号） |
| POST | `/accounts` | 新建账号（仅资料，不含 token） |
| GET | `/accounts/{id}` | 详情 |
| PUT | `/accounts/{id}` | 更新地址、策略、备注 |
| DELETE | `/accounts/{id}` | 删除 |
| POST | `/accounts/{id}/send-vcode` | 触发发码（调 i茅台） |
| POST | `/accounts/{id}/login` | 提交验证码完成登录，Token 入库 |
| POST | `/accounts/{id}/validate-token` | 检测 Token 是否有效 |
| POST | `/accounts/import` | CSV/Excel 批量导入 |
| GET | `/accounts/{id}/status` | 登录状态、最后预约时间、最后错误 |

**POST `/accounts/{id}/login` 请求：**

```json
{
  "vcode": "123456"
}
```

**响应 data：**

```json
{
  "user_id": "12345678",
  "token_valid": true,
  "logged_at": "2026-05-20T08:00:00+08:00"
}
```

**账号模型字段（数据库）：**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| mobile | string | 手机号 |
| user_id | string | i茅台 userId |
| token_enc | string | 加密 Token |
| device_id | string | 设备 ID |
| province / city / lat / lng | string | 地区 |
| receiver_* | string | 收货信息 |
| shop_strategy | enum | max_inventory / nearest / fixed |
| fixed_shop_id | string? | 绑死门店 |
| enabled | bool | 是否参与批量任务 |
| end_date | string | 账号有效期 |

### 3.6 预约任务 ` /jobs `（合同 (3)）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/jobs` | 任务列表（定时/手动/历史） |
| POST | `/jobs` | 创建任务 |
| GET | `/jobs/{id}` | 任务详情 + 进度 |
| POST | `/jobs/{id}/cancel` | 取消排队中任务 |
| POST | `/jobs/{id}/run` | 立即执行（等同 CLI 正式预约） |
| POST | `/jobs/dry-run` | 试跑（不提交） |

**POST `/jobs` 请求：**

```json
{
  "name": "每日申购-早场",
  "type": "scheduled",
  "cron": "0 58 8 * * *",
  "account_ids": [1, 2, 3],
  "product_ids": [1, 2],
  "dry_run": false
}
```

**任务状态枚举：** `pending` | `running` | `success` | `partial` | `failed` | `cancelled`

**记录表字段：**

| 字段 | 说明 |
|------|------|

## 十、与一期 CLI 的关系

| 能力 | 一期 CLI | 二期 API |
|------|----------|----------|
| 使用方式 | 本机终端 | 浏览器 + 服务端 |
| 数据存储 | `credentials.json` | MySQL 加密 |
| 定时 | launchd / 手动 | 服务端 Cron + 任务队列 |
| 适合 | 个人自用 | 合同验收、多账号运维 |

**一期 CLI 可继续独立使用**；二期上线后 CLI 可改为调用 `http://localhost:8000/api/v1`（可选）。
